# Customer Credit Card Delinquency Dashboard

## Project Overview
This Power BI dashboard analyzes customer credit card data and provides insights into customer distribution, credit scores, age patterns, and account tenure trends.

## Tools Used
- Power BI
- Excel

## Dashboard Insights
- Distribution of customers by credit card type
- Credit score comparison across card types
- Customer age distribution
- Credit score trends by account tenure
